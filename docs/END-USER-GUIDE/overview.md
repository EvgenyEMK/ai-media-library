# What this app is for

EMK Desktop Media helps you organize and explore your local photo/video library with AI-assisted features.

## Key points for non-technical users

- Your library is desktop-first and local-first.
- AI features help you find and understand media faster.
- You can use classic browsing and filtering even without advanced AI setup.
- Some AI features depend on external model runtimes or model files, which have their own licenses.

## How it is different from a typical media app

- Strong focus on desktop workflows and local libraries.
- Architecture designed to share core packages between desktop and web clients.
- AI-focused experience for semantic search, metadata enrichment, and face/person workflows.
